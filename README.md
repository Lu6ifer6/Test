
# OCTOM. Web Application

Welcome to the OCTOM. project! This repository contains the front-end code for a modern, responsive web application featuring a stunning landing page, an interactive user dashboard, and a comprehensive settings panel.

## Features

Landing Page: A beautiful, fully responsive entry page with a hero section, feature highlights, a responsive customer review carousel, and a detailed footer.

Dashboard: A data-rich dashboard including interactive data visualizations using Chart.js, task tracking, and statistical summary cards.

Settings Interface: A user profile management page with a cover photo, avatar setup, and drag-and-drop file upload UI.

Responsive Design: Fluid layouts that adapt perfectly to desktop, tablet, and mobile displays. Features an interactive sidebar with a mobile-friendly hamburger menu overlay.

## Technologies Used

HTML5: Semantic and accessible markup.

CSS3: Advanced styling using CSS Custom Properties (Variables), CSS Grid, Flexbox, and mobile-first media queries.

Vanilla JavaScript: Clean DOM manipulation for sidebar toggling, event handling, and mobile overlay management.

Chart.js: Rendered interactive line charts for tracking task/project data in the dashboard.

## Project Structure

`/css/` - Contains modular stylesheets (`style.css` for the dashboard layout, `settings.css` for the user settings page).

`/javascript/` - Contains modular scripts (`dashboard.js` for charts and menus, `settings.js` and `files.js` for UI logic).

`style.css` - Global and landing page-specific styling (Hero, Features, Reviews).

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Lu6ifer6/Test.git
   ```
2. Open the project:
   Simply open the `index.html` file (or the respective HTML files for the dashboard and settings) in your preferred web browser. No build tools or local servers are required to view the static pages!
